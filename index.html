About the Project

This project built in Python for managing and tracking personal expenses. Allows users to categorize, add, and save their expenses, as well as view an expense report.

Features

  • Expense Categorization
  • Add Expense
  • Save Expense Report to CSV
  • Menu-Driven Interface
  • Built with Python libraries like Pandas and NumPy
